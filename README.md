# net-scanner: Набор утилит для сетевого анализа

Коллекция независимых скриптов на Python для автоматизации задач системного администрирования: инвентаризации сети и целевого аудита узлов.

## Состав набора

Проект разделен на два автономных модуля для решения разных задач:

### 1. `arp.py` — Сетевая разведка
Автоматизированный инструмент для сканирования локального сегмента сети без ручного ввода адресов.
*   **Принцип работы:** Извлекает данные из системной таблицы ARP (`arp -a`).
*   **Фильтрация:** Интеллектуальный парсинг исключает мультикаст-адреса (224.x.x.x), широковещательные запросы (.255) и системные заглушки.
*   **Аудит:** Автоматически пингует найденные живые узлы и сканирует стандартные порты (21, 22, 80, 443, 3389, 8080).

### 2. `main.py` — Целевой сканер
Интерактивный инструмент для проверки конкретных узлов или списков IP.
*   **Гибкость:** Поддерживает ввод одного IP или списка через запятую.
*   **Валидация:** Проверка корректности введенных данных с помощью регулярных выражений перед началом работы.

## Технические подробности
*   **L3 Анализ:** Проверка доступности через ICMP (Ping) с верификацией поля `TTL` для корректной работы в ОС Windows.
*   **L4 Анализ:** Сканирование TCP-портов через библиотеку `socket` с таймаутом 0.3с.
*   **Zero Dependencies:** Инструменты используют только стандартную библиотеку Python.

## Инструкция по использованию

1. **Клонирование проекта:**
   ```bash
   git clone https://github.com
   cd net-scanner

   ```
2. **Для сканирования конкретного адреса или пула адресов запустить main.py и ввести адрес(-а):**
	```bash
		python main.py
	```
3. ** Для запуска сканирования arp таблицы запустить arp.py:
	```bash
		python arp.py
	```
